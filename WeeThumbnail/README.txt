WeeThumbnail - A thumbnail handler for displaying Ragnarok Online sprites

Use this version only on Windows XP SP2 or higher.
You will find two directories inside this folder.

x86:
------------
This one contains only the binary for 32 bit windows versions.
It will, however, copy the *.dll to their proper directories.

%SystemRoot%\system32 on 32 bit
%SystemRoot%\SysWOW64 on 64 bit

Use this one only if you're running windows 32 bit or have problems
getting the one below to work.

x86 and x64:
------------

This one contains additionally a binary compiled for 64 bit windows.
The batch file will copy the proper *.dll to %SystemRoot%\system32.
